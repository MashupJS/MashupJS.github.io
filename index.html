one code base

With the modern web, it is possible to build a single code base capable of satisfying all platforms.

It's no longer necessary to build a separate code base for the web, iOS, Android, and Windows Phone.

Don't lock your company into a mobile hardware platform. Deliver code that can run on any mobile platform the business chooses.

The MashupJS is a bootstrap for building one code base.
